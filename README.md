•	Дата и время вылета (факт)
•	Дата и время прилета (факт)

•	Задержка прилета (разница между фактической и запланированной датой в секундах)
•	Самолет
•	Аэропорт вылета
•	Аэропорт прилета
•	Класс обслуживания
•	Стоимость
